# Artifact Center

面向企业内网的现代化制品（Artifact）管理平台。

用于管理 Android APK / AAB、Windows EXE、ZIP 等构建产物；体验对标 Linear、GitHub、Vercel 等现代产品，而非传统 ERP 后台。

## 当前阶段

**P0 Foundation** — 仅项目规范与文档，尚未初始化应用工程。
